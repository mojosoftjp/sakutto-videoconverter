さくっと動画変換SakuttoVideo

はじめに

「さくっと動画変換」（以下「本アプリ」）は、ユーザーのプライバシーを最大限に尊重して設計されています。本ポリシーでは、本アプリにおける情報の取り扱いについて説明します。

個人情報の収集について

本アプリは、ユーザーの個人情報を一切収集しません。

  • 氏名、メールアドレス、電話番号などの個人情報は収集しません
  • 利用状況やアクセスログなどの分析データは収集しません
  • Cookie やトラッキング技術は使用しません

データの処理について

本アプリで行われるすべての動画処理（フォーマット変換・トリミング・音声抽出・圧縮など）は、ユーザーのPC上でのみ実行されます。

  • ファイルが外部サーバーに送信されることはありません
  • インターネット接続なしで全機能を利用できます
  • 処理対象のファイルは、ユーザーが指定した保存先にのみ出力されます

使用しているオープンソースソフトウェア

本アプリは動画処理エンジンとして FFmpeg (LGPL版) を使用しています。FFmpegのソースコードは https://ffmpeg.org/ で入手できます。

ネットワーク通信について

本アプリはネットワーク通信を行いません。すべての処理はローカル環境で完結します。

第三者への提供

収集するデータが存在しないため、第三者へのデータ提供は一切ありません。

お子様のプライバシー

本アプリは個人情報を収集しないため、お子様を含むすべてのユーザーに安全にご利用いただけます。

本ポリシーの変更

本ポリシーは、必要に応じて更新される場合があります。変更があった場合は、本ページにて更新日とともに公開します。

Introduction

"SakuttoVideo" (hereinafter "the App") is designed with the utmost respect for user privacy. This policy explains how information is handled within the App.

Collection of Personal Information

This App does not collect any personal information.

  • No personal data such as names, email addresses, or phone numbers is collected
  • No usage analytics or access logs are collected
  • No cookies or tracking technologies are used

Data Processing

All video processing performed by the App (format conversion, trimming, audio extraction, compression, etc.) is executed entirely on the user's PC.

  • Files are never sent to external servers
  • All features work without an internet connection
  • Processed files are saved only to the location specified by the user

Open Source Software Used

This App uses FFmpeg (LGPL version) as its video processing engine. FFmpeg source code is available at https://ffmpeg.org/.

Network Communication

This App does not perform any network communication. All processing is completed locally.

Third-Party Sharing

Since no data is collected, there is no data sharing with third parties.

Children's Privacy

Since the App does not collect personal information, it is safe for all users, including children.

Changes to This Policy

This policy may be updated as needed. Any changes will be published on this page with the updated date.
